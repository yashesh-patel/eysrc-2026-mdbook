# Task 5: Overview

---

## Welcome to Task 5

Welcome to the **final challenge** of the Disaster Management theme.

In this mission, the **e-puck** robot is deployed inside a partially collapsed underground railway terminal where stranded train cars must be rescued before time runs out. Unlike the previous tasks, this challenge requires the robot to integrate **multiple autonomous behaviours** into a single rescue mission.

The robot must first navigate a maintenance tunnel using **wall following**, detect and decode **ArUco markers** to determine the rescue priority, and then transition into the railway network where it performs **line following**, navigates intersections, and rescues train cars in the correct order.

Task 5 combines all the concepts learned throughout the course into one complete autonomous rescue system.

---

## What You Will Learn

- ArUco Marker Detection
- Sequential Memory

- Intersection Navigation

- Complete Autonomous Navigation

## Why This Task Matters

Real-world autonomous robots rarely perform a single task. Instead, they combine perception, navigation, memory, and decision-making to solve complex real-world problems.

Task 5 challenges you to integrate all the techniques learned throughout the course into one intelligent rescue robot.
