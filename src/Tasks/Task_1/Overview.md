# Task 1: Overview

---

## Welcome to Task 1

Task 1 introduces the fundamentals of **autonomous robotics** using the Webots simulation environment.

In this task, you will program the **e-puck** robot to detect coloured targets using its onboard camera, navigate towards them, and collect the corresponding supply boxes autonomously.

This task lays the foundation for all the upcoming challenges.

---

## What You Will Learn

- Camera programming in Webots
- OpenCV image processing
- HSV colour detection
- Robot motion control
- Basic autonomous navigation

---

## Learning Flow

```text
Learn → Build → Test → Submit
```

---

## Why This Task Matters

Visual perception is one of the most important capabilities of an autonomous robot. The concepts introduced in this task will prepare you for advanced topics such as line following, ArUco detection, wall following, and autonomous decision-making in the upcoming tasks.
