<h1 style="text-align: center;">Dictionaries</h1>

<hr>

Unlike other data types discussed so far that hold only a single value as an element, a **Dictionary** holds data as **key:value** pairs.

In Python, a dictionary can be created by placing a sequence of elements within curly braces `{}`, separated by commas.  
Each element consists of a **key** and its corresponding **value** — together forming a **key:value** pair.

Values in a dictionary can be of any data type and can be duplicated, whereas keys **cannot be repeated** and must be *immutable*.

> [!IMPORTANT]
> Dictionaries were modified to maintain insertion order starting with Python 3.7.  
> Hence, they are now an **ordered collection** of data values.
