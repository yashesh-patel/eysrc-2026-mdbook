<h1 style="text-align: center;">Data Types</h1>

<hr>

Variables can store data of different types and based on that interpreter allocates  memory and decides what can be stored in the reserved memory. different types can do different things.

For example, a student's name is stored as string, his roll number is stored as integer, percentage can be stored as float. Python has various standard data types that are used to define the operations possible on them and the storage method for each of them.

Python supports the following built-in data types:

1. Integer
2. Float
3. Complex Numbers
4. String
5. List
6. Tuple
7. Dictionary
