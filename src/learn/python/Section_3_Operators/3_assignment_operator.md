<h1 style="text-align: center;">Assignment Operators</h1>

<hr>

Assignment operators are used to assign values to variables.

1. **Equal or Assign (`=`)** : You have seen that a single equal sign (`=`) is used to assign a value to a variable. It assigns the value of the right-side operand to the left-side operand.

   <iframe src="https://trinket.io/embed/python3/803d23fc7b" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

2. **Add and Assign (`+=`)** : Adds two numbers and assigns the result to the left operand. `a += b` is the same as `a = a + b`.<br>

   <iframe src="https://trinket.io/embed/python3/271ecf9ac0" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

3. **Subtract and Assign (`-=`)** : Subtracts two numbers and assigns the result to the left operand. `a -= b` is the same as `a = a - b`.

   <iframe src="https://trinket.io/embed/python3/0e29759fbe" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

4. **Multiply and Assign (`*=`)** : Multiplies two numbers and assigns the result to the left operand. `a *= b` is the same as `a = a * b`.

   <iframe src="https://trinket.io/embed/python3/6021a45593" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

5. **Division And Assign (`/=`):**  Divides two numbers and assigns the result to the left operand. `a /= b` is the same as `a = a / b`.

   <iframe src="https://trinket.io/embed/python3/f372a29238" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

6. **Modulus and Assign (`%=`):** Computes the modulus of two numbers and assigns the result to the left operand. `a %= b` is the same as `a = a % b`.

   <iframe src="https://trinket.io/embed/python3/3ffa104d66" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

7. **Exponent and Assign (`**=`):** Performs exponentiation (power calculation) and assigns the result to the left operand. `a **= b` is the same as `a = a ** b`.

   <iframe src="https://trinket.io/embed/python3/b855a43f30" width="100%" height="356" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>
