<h1 style="text-align: center;">Comparison (Relational) Operators</h1>

<hr>


A condition is a programming statement that compares values and tells us whether the criteria set by the comparison are **True (yes)** or **False (no)**. For example, let's say we have two chocolate boxes, i.e., `box1` and `box2`.

1. **Equal To (==)** : If there are 20 chocolates in each box, the condition `box1 == box2` would return **True**; otherwise, it would return **False**. 

2. **Not Equal To (`!=`)**: If there are 10 chocolates (Example 2) in `box1` and 20 in `box2`, then the condition `box1 != box2` would return **True**.

3. **Greater Than (`>`)**: If we consider the same condition given in (Example 2), then the condition `box1 > box2` would return **False**, as `box1` contains fewer chocolates compared to `box2`.

4. **Less Than (`<`)**: Here, the condition `box1 < box2` would return **True**.

5. **Less Than or Equal To (`<=`)**: `box1 <= box2` would return **True**, as there are 10 chocolates, which is less than or equal to the 20 chocolates in `box2`.

6. **Greater Than or Equal To (`>=`)**: `box1 >= box2` would return **False**, as 10 chocolates are not greater than or equal to the 20 chocolates in `box2`.
