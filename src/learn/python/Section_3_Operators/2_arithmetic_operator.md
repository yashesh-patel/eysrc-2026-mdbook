<h1 style="text-align: center;">Arithmetic Operators</h1>

<hr>

Arithmetic operators are used to perform basic mathematical calculations, such as addition, subtraction, multiplication, division, and modulo operations.

#### For example: 

1. **Addition (`+`)** : Returns the sum of `a` and `b`.

2. **Subtraction (`-`)** : Returns the difference of `a` and `b`.

3. **Multiplication (`*`)** : Returns the product of `a` and `b`.

4. **Division (`/`)** : Returns the quotient of `a` divided by `b`.

5. **Modulus (`%`)** : Returns the remainder when `a` is divided by `b`.

6. **Exponent ( `**` )** :` a ** b` returns `a` raised to the power of `b`.

7. **Floor Division (`//`)** :  Divides `a` by `b` and returns the integer value of the quotient by removing the digits after the decimal point. If one of the operands is negative, the result is floored, i.e., rounded away from zero (towards negative infinity).
