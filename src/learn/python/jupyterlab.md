# <center>Getting Started with JupyterLab</center>
<hr>

JupyterLab is an web-based interactive Python code editor which will help you to keep track of your code more easily. 

To practice online the different topics covered in the Python Tutorials you can use JupyterLab. Go to this [link](https://jupyter.org/). The video below shows  `How to get started with JupyterLab`.

To practice locally on your system you can install Jupyter Notebook. The installation instructions for setting up Jupyter Notebook is given below:

<b>Installation Instructions for setting up Jupyter Notebook using pip:</b>

- Open Command Prompt/Terminal and type `pip install jupyterlab`.
- Once installed, yo run the jupyter notebook, run the following command at the Terminal (Mac/Linux) or Command Prompt (Windows): `jupyter-lab`.
