# Python Programming

Welcome to the Python Programming section! Python is the primary language used to program the e-puck robot in Webots. It is known for its simplicity and readability, making it an excellent language for robotics.

---

## Why Python for Robotics?

Python provides extensive libraries for computer vision (like OpenCV), mathematical computations (like NumPy), and data handling. Its integration with Webots allows you to quickly write, test, and iterate on your robot controllers.

---

## Python Learning Modules

The learning material is divided into sections. If you are new to programming or Python, we strongly recommend going through these modules:

1. **Computer Organisation** - Understanding how code runs on a machine.
2. **Python Basics** - Variables, data types, and basic syntax.
3. **Operators** - Arithmetic, logical, and comparison operators.
4. **Data Structures** - Lists, tuples, dictionaries, and sets.
5. **Control Flow (Part 1)** - If-else statements and basic decision making.
6. **Control Flow (Part 2)** - Loops (for, while) to execute repetitive tasks.
7. **Functions** - Creating reusable blocks of code for your robot.

*Note: The detailed exercises and reading materials for these sections can be found in the respective folders in the source repository.*

---

## Getting Started

Before diving into the code, ensure you have set up your environment correctly:

- Have you installed Python 3? (Refer to the [Software Installation](../WebotsTutorials/Software_Installation.md) page).
- Have you set the Python executable path in Webots Preferences?

Once your setup is complete, you can start writing your first Webots controller!

> **NOTE!** <span style="color:red">Indentation is critical in Python. A missing or extra space can cause your controller to fail. Always ensure your code is properly aligned.</span>
