## Computer Program
* A computer program is a collection of instructions that can be executed by a computer to perform a specific task.
* A computer is a hardware that doesn't know what to do with itself if it isn't given instructions.
* A computer program is usually written by a computer programmer in a programming language.
* A program can be as complex as an operating system or as simple as a calculator.
* A collection of computer programs, libraries, and related data are referred to as software.


## Computer Programming
* Computer programming is the process of designing and building a computer program to compute a result or to perform a specific task.
* The purpose of programming is to find a sequence of instructions that will automate the  task on a computer, often for solving a given problem.


## Programming Languages
* Instructions to the computer have to be given in a language they understand. Such languages are called programming languages. 
* There are around 500+ programming languages in the computer world, each with its unique syntax and functionalities.
*  Ada Lovelace created the first computer programming language.
* A programming language is a formal language comprising a set of instructions that converted to machine code. Machine code is a sequence of 1s and 0s that is understood by the machine.
* Programming languages are one kind of computer language, and are used in computer programming to implement algorithms.
