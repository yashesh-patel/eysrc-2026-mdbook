## How to Use the Task Package

After downloading the ZIP file for a task:

1. Extract the ZIP to a folder on your computer.
2. Open **Webots**.
3. Go to `File > Open World`.
4. Navigate to the extracted folder and open the `.wbt` file inside the `worlds/` directory.
5. Open the controller file from the `controllers/` directory in your code editor.
6. Write your solution and test it in Webots.
7. Once done, run the simulation to completion - a submission ZIP will be auto-generated.
