Welcome!

Robots are becoming an essential part of our daily lives—from assisting in industries and healthcare to exploring environments that are difficult or dangerous for humans. Before deploying a robot in the real world, engineers often test and validate their ideas in a simulation environment. This is where **Webots** plays an important role.

In this competition, you will learn the fundamentals of autonomous robotics by programming the **e-puck** robot in Webots. Through a series of interactive tutorials and themed challenges, you will gradually build the skills required to develop intelligent robotic behaviours using **Python**, **Image Processing**, and **computer vision**.

The learning journey has been designed around a disaster-response theme. Each task introduces a new robotics concept while contributing to an overall rescue mission. Starting with basic camera-based color detection, you will progressively explore line following, shape detection, ArUco markers, state machines, and wall following before combining these concepts in the final challenge.

---

## What You Will Learn

Throughout this course, you will gain hands-on experience with:

- Python Programming

- e-puck Robot Programming
- OpenCV Image Processing
- Color Detection
